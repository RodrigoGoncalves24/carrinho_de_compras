<h1>Carrinho de compras:</h1>

<h2>Sobre</h2>
<p>Projeto desenvolvido durante o curso de prática em lógica de programação fornecidas pela Alura. Este projeto tem como objetivos a criação de uma replica de site de compras
  onde o usuário poderá adicionar itens ao carrinho juntamente com a quantidade desejada.</p>

## Como usar
  Ao entra no site o usuário contará com um campo de escolha para os pedidos do qual pedirá para relizar a escolha de um produto para inserir no carrinho de compras,
  conforme o usuário for informando os pedidos, o programa irá calcular o preço total dos pedidos. O usuário tem a possibilidade de limpar o carrinho também e isso irá excluir todos os itens existentes. 

  Boas compras!!
